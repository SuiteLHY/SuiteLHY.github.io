Thymeleaf 是一个 Java XML / XHTML / HTML5 模板引擎，可以在 Web（基于 servlet ） 和 非 Web 环境中 工作。它更适合 在基于 MVC 的 Web 应用程序的视图层 提供 XHTML / HTML5，但它甚至可以 在脱机环境中 处理任何 XML 文件。它提供完整的 Spring Framework。

在 Web 应用程序中，Thymeleaf 旨在成为 JavaServer Pages（JSP）的完全替代品，并实现 自然模板的概念：模板文件 可以直接在浏览器中 打开，并且 仍然可以 正确显示为网页。

Thymeleaf 是 开源软件、许可下 Apache 许可证2.0。
